Adoption workflow

Optimize, Evaluate, And Approve Skills

Follow a structured workflow to ensure skills bring reliable guardrails and evals.

Flow
1

Use skill-creator

Start from the existing skill and run the same refinement workflow used for a new skill.

2

Attach the new or existing skill

Give the workflow the real skill package, not an abstract description.

3

Attach the checklist

Set explicit review criteria for structure, guardrails, and evidence.

4

Strengthen eval support

Refine the description and stop gates while adding the missing eval coverage.

Result

Benchmarked, reviewable, approval-ready

The outcome is a skill package that carries eval support, the review rubric, and clearer approval evidence together.
