Review Outputs And Benchmark Data Together

Teams need qualitative output review and quantitative benchmarks to assess skill value.

Outputs
  • Prompt text and run context for the selected eval case
  • Generated files rendered inline where possible
  • Formal grades from grading.json
  • Previous-output comparison when the prior iteration exists
Reviewer input
  • Feedback box that persists into feedback.json
  • Notes about ambiguity, variance, or low-value assertions
  • Human judgment layered on top of formal grading
